** 펼침 여산자 (Spread Operator) **
		=> 배열의 원소 또는 객체의 속성 등을 펼쳐서 할당
		=> 즉, 배열의 원소 또는 객체의 속성 등을 모두 한번에 다른 배열 또는 객체에 포함 시킬 수 있음
		=> 함수의 매개변수로 활용하여 한번에 인자를 전달 받을 수 있음.
		
		=> 함수 인자나 배열 등이 연속적으로 나오는 곳이면 어디든 쓸 수 있는 유용한 연산자
		=> 펼침 연사자의 표기방법은 '...' 
		
		   var params = [ "hello", true, 7 ]
		   var other = [ 1, 2, ...params ] // [ 1, 2, "hello", true, 7 ]
		   f(1, 2, ...params) === 9
		   
		   var str = "foo"
		   var chars = [ ...str ] // [ "f", "o", "o" ]
		   
		=> arguments : 배열과 같은 형태로 매개변수값을 전달히는 객체 (펼침 연산자 적용가능)
