** window.location Test **

        => window의 내장 객체 : 윈도우에 로드된 웹 페이지의 URL 정보를 나타냄
        => location : window.location=url (url로 이동, 히스토리에 기록됨)
        => location 의 메서드 (jq02_03countDown.html 참고)
           ->2.1 href : 속성, 화면전환 (히스토리에 기록 -> 이전페이지로 이동가능)
           ->2.2 replace('...') : 화면전환 (히스토리에 기록안됨 -> 이전페이지로 이동불가능)
           ->2.3 reload() : 일반적인 새로고침, (f5)
                      컴퓨터의 캐쉬에서 우선 찾고 없으면 서버에서 받아옴
           ->2.4 reload(true) : 강력한 새로고침
                           캐쉬는 무시하고 무조건 서버에서 받아옴
