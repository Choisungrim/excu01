     ** 자동 화면전환 **
     1) HTML : meta Tag
        -> http-equiv="refresh" content="5;url=http://www.naver.com"
           5초후 Naver 로 이동   
     
     2) JS : location 객체 (window.location...) 
        -> href_속성 : 화면전환 (히스토리에 기록 -> 이전페이지로 이동가능)
        -> replace('...') : 화면전환 (히스토리에 기록안됨 -> 이전페이지로 이동불가능)
        -> reload() : 새로고침 (f5)
